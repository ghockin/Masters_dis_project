# Masters_dis_project
A python based project where OCR or Pdf Plumber is used to scan a pdf and model &amp; simulate a scenario, using Machine learning to develop its own new methods through the process of Reinforcement learning.


Pip installs needed:
pdf2image          1.17.0
Flask              3.1.3
pdfplumber         0.11.9
pillow             12.1.1
